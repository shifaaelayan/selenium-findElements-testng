# Selenium Find Elements Practice

This project is a simple Selenium automation practice using Java and TestNG on the SauceDemo website.

## What this project covers
- Login automation using Selenium WebDriver
- Using `findElements()` to handle multiple elements
- Working with CSS selectors for elements with multiple classes
- Adding all products to the cart using a loop
- Removing products after adding them
- Using JavaScript alerts during test execution
- Repeating tests using TestNG XML instead of `invocationCount`

## Tools & Technologies
- Java
- Selenium WebDriver
- TestNG
- Edge Driver

## Test Execution
The same test class is executed multiple times using `testng.xml` to practice test repetition and execution flow.

## Website Used
https://www.saucedemo.com/

## Purpose
This project is created for learning and practicing Selenium automation concepts, element handling, and TestNG execution strategies.
